Js-Scrollbox that scroll images of gray-scale and showing original image on hover. 

== Description ==

This Plugin is based on jQuery and CSS and Working on jQuery.This Plugin is useful for Users who Want a Scrollbox like Slider.
This Plugin is work like a scrollbox that scroll the images with in the box horizontally.I used a custom post type for that plugin and it generated a shortcode using post id.I am provide a hover image property too so the image is hover with the other image too.This plugin is responsive too.

It is based on Light jquery.

**Plugin Feature **

This Plugin is light jquery based that scroll multiple images

1. It support Multiple images.
2. It provide Hover Image option too so if you want than your image hover with other image.
3. At This time it provide maximum ten images and it is responsive too.

** How To Use it**

In this Plugin We have a Add Media Button at top Left cornor of editor Using that 
you can pick up that path of required images so you don't need to go media Library 
again and again


if you find any bug or problem then feel free to ask me. I am happy to solve it.   

== Installation ==

1. Download the latest version and extract it in the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

To disable the shortcode, deactivate the plugin in the 'Plugins' menu in WordPress.
